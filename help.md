
Функции:  

`print()` - выводит значение  
`input()` - берёт с клавиатуры строку  


`int()` - переводит строку в ЦЕЛОЕ число  
`float('12.5')` - переводит строку в число с запятой  
`str()` - из числа в строку  


`chr()` - возвращает символ по его номеру  
`ord()` - возвращает порядковый номер символа  
`len()` - возвр длину строки  

`s[m:n]` - извлекает из строки подстроку  
- срез строки от m до n, не включая n  

---  

Операции сравнения:  

- могут возвращать True или False (логический тип данных)  

`>`   - больше  
`==`  - равно  
`!=`  - не равно  
`>=`  - больше или равно  

Операции арифметические:  

`+ - * /`
