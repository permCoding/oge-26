## задание 14 ОГЭ по информатике  

### Основные формулы для задания 14  

| Что нужно найти | Какая формула нужна | Пример задания |
| :--- | :--- | :--- |
| **Сумму чисел** | `=СУММ(диапазон)`<br>`=СУММЕСЛИ(диапазон_условия; "условие"; диапазон_суммирования)` | Найти общее количество баллов всех учеников. <br> Найти общее количество баллов только отличников. |
| **Среднее арифм.** | `=СРЗНАЧ(диапазон)`<br>`=СРЗНАЧЕСЛИ(диапазон_условия; "условие"; диапазон_усреднения)` | Найти средний балл среди всех учеников. <br> Найти средний балл среди выбравших информатику. |
| **Кол-во значений** | `=СЧЁТ(диапазон)` — считает только числа<br>`=СЧЁТЗ(диапазон)` — считает все ячейки, включая текст<br>`=СЧЁТЕСЛИ(диапазон; "условие")`<br>`=СЧЁТЕСЛИМН(диапазон1; "условие1"; диапазон2; "условие2")` | Найти кол-во учеников, набравших больше 80 баллов. <br> Найти кол-во отличников, которые учатся в школе №5. |
| **Макс/мин значение** | `=МАКС(диапазон)`<br>`=МИН(диапазон)` | Найти лучший и худший результат в классе. |
| **Макс/мин по условию** | `=МАКСЕСЛИ(диапазон для max; условие; диапазон для условия)`<br>`=МИНЕСЛИ(диапазон для max; условие; диапазон для условия)` | Найти максимальный или минимальный результат по условию отбора. |
| **Логическое условие** | `=ЕСЛИ(условие; значение_если_истина; значение_если_ложь)` | Поставить "Зачет"/"Незачет" в зависимости от набранных баллов. |

Чаще всего встречаются задачи с **одним или двумя условиями отбора**, например, найти средний балл учеников из Северного округа, сдавших информатику (два условия отбора: округ и предмет)  

### Часто используемые формулы  

1. `=СЧЁТЕСЛИМН(A2:A1001; "В"; C2:C1001; "информатика")` - сколько учеников из округа "В" выбрали информатику - много проверяемых условий в одной формуле  
2. `=СРЗНАЧЕСЛИ(A2:A1001; "С"; D2:D1001)` - средний балл всех учеников из округа "С"  
3. `=ОКРУГЛ(__, 2)` — ответ с точностью до двух знаков после запятой  

#### ВАЖНО  

- любые условия отбора или сравнения в формулах следует писать в кавычках, например, `английский язык` или `>`  
- если есть необходимость использовать ссылки на конкретные ячейки, то следует на них ставить `абсолютные ссылки` (не относительные), чтобы во время копирования ссылки не сдвигались (относительная ссылка на ячейку пишется так: `A2:A1001`, абсолютная - со знаком `$` - так: `$A$2:$A$1001`)
